---

## 📝 技術規格

- **純前端實作**：HTML + CSS + JavaScript，無需後端伺服器
- **Web Audio API**：內建音效合成（答對/答錯/升級/攻擊/開寶箱）
- **SVG 向量素材**：音符與指法圖無限縮放不失真
- **響應式設計**：支援各種螢幕尺寸
